# 五子棋在线对战平台

一个支持本地对战和在线对战的五子棋游戏，具有现代化的界面设计和丰富的功能。

## 功能特性

### 游戏模式
- **本地对战**：双人同屏对弈，支持自定义玩家名称
- **在线对战**：创建房间或加入房间，与好友远程对弈

### 游戏功能
- ⏱️ **计时系统**：支持 3/5/10 分钟或不限时
- 🔄 **悔棋功能**：可设置悔棋次数限制（1/3/5次/无限）
- 🏆 **比赛模式**：支持单局决胜、三局两胜、五局三胜
- 💬 **快捷消息**：预设消息快速发送
- 😀 **表情系统**：丰富的表情互动

### 棋盘特性
- 📐 **标准规则**：棋子精确落在网格线交点上
- ⭐ **星位标记**：显示天元和边星位置
- 📱 **响应式设计**：自适应屏幕大小，手机端完美显示
- 🎨 **视觉效果**：五子连珠高亮动画、最后落子标记

### 对局记录
- 📋 **历史记录**：保存最近 50 局对局信息
- 📸 **导出棋谱**：导出带序号的棋盘图片（PNG 格式）
- 🔢 **落子序号**：游戏结束后显示每步棋的顺序

## 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript
- **后端**：Node.js + WebSocket
- **部署**：GitHub Pages（前端）+ Railway（后端）

## 本地运行

### 前端
直接打开 `index.html` 即可运行本地对战模式。

### 后端（在线对战）
```bash
cd server
npm install
npm start
```

服务器默认运行在 `ws://localhost:8080`。

## 游戏规则

1. 黑方先行，双方轮流落子
2. 先将五颗棋子连成一线（横、竖、斜）者获胜
3. 支持悔棋（需对方同意）
4. 超时判负
